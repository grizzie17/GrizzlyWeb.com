Configure Multiple Computers to Read Single E-mail

Many people now have multiple computers and want to access a single e-mail account.  The biggest problem is that normally when you access your email on a server with an e-mail client such as Outlook it will clean the e-mail from the server.  Which means that when you go to your other computer and attempt to access your e-mail you find that it is missing.  The secret is to leave your e-mail on the server and carefully control when it is cleaned up.

We will create two terms to refer two types of configurations.

  • Manager - This configuration will manage the e-mail on the server (don't worry its real simple)
  • Client - This configuration will always leave the e-mail on the server

You need to make sure that you have only one manager.  It is also important that your manager configuration of Outlook is one that you are going to use on a regular basis.  When you have made your decisions about which machines are to be configured.  The only difference between them is found on the "More Settings..." under the "Advanced" tab.

Client Configuration

Simply select the "Leave a copy of messages on the server" checkbox under the "Delivery" heading.

Manager Configuration

For the manager configuration select the "Leave a copy of messages on the server" under the "Delivery" section.

Select "Remove from server after N days".  I suggest keeping the number of days between 3 and 14.
