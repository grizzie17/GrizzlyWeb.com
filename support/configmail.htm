Select the "Start" menu from the task-bar at the bottom of the screen, then select "Settings", and then select "Control Panel" from the hierarchical menu.

From the Control Panel folder select the "Mail" icon.

The "Internet Accounts" dialog should be displayed.  On some systems it may directly invoke the mail account wizard.

Select the "Add..." button.

Enter a name.  Don't worry too much about what you put here because we are going to change it later.

When you have entered a name, select the "Next>" button.

Enter the email address on your domain that you want to be able to use from within Outlook 2000.

Select the "Next>" button.

Make sure the first item on this dialog is "POP3".  Enter "mail.YourDomain.com" for the "Incoming mail (POP3 or IMAP) server".  For instance if your domain's name is "PetesStuff.com", you would enter "mail.PetesStuff.com".  For the "Outgoing mail (SMTP) server enter the mail server for your ISP (Internet Service Provider).  That is the mail server that you use where you dial-in.  Your ISP should have already provided you with this information.

Select the "Next >" button.

Enter your account name on YourDomain.  This will be the name that you specified in front of the at-sign "@".  Then specify the password of your email on YourDomain.  This information will be provided to you from Bear Consulting Group.

Select the "Next >" button.

Select "Connect using my phone line" if you use a modem to connect to the internet.  If you have a hard-line to the internet then select "Connect using my local area network".  Don't bother selecting the last option.

Select the "Next >" button.

Since you should already have a dial-up connection (via your ISP) select the connection that is correct for you ("My Connection" is just a placeholder for this picture of a dialog).

Select the "Next >" button.

This will bring up the "Finish" dialog (which has no input).  Select the "Finish" button.

There should now be a new account showing in the "Internet Accounts" dialog with the name "mail.YourDomain.com".

Select the newly created account and then select the "Properties" button.

Under the "General" tab; select the "Mail Account" and replace it with something more meaningful to you.  For instance if you were creating this account to read your sales account from YourDomain you might want to name it "Sales - YourDomain".  Don't worry about it though, because you can always come back to this dialog to change it later.

Add the "Reply address" as the same as the E-mail address.

Under the "Servers" tab; make sure that it looks similar to the above dialog (obviously with your information instead of my example information).

Under the "Connection" tab; make sure it looks similar to the example dialog above (with the exception that your dial-up connection will be the one listed).

Under the "Advanced" tab; don't bother with anything on this dialog except for the "Leave a copy of messages on server" check box under "Delivery".  If you have more than one computer connecting to YourDomain to check the specified account you may want only one of the computers to actually remove the messages from the server.  You see if the "Leave a copy of messages on server" is not checked whenever you check your mail all of the messages will be copied to your computer and removed from the server.  Then if someone else checks from a different computer they will not be able to see the messages that you read.  Think carefully about this before setting it one way or the other.

Finally select the "OK" button.

You have now added an email account that will check YourDomain mail messages.  If you have other YourDomain mail accounts that you need or want to check, repeat the previous steps.

If you haven't already done so, you will also want to add an account for your standard ISP mail account.

If you have created multiple accounts, in the "Internet Accounts" dialog you will want to select the account that you want to use by default to send messages with Outlook 2000.  Simply select the account, and then select the "Set As Default" button.

Finally select the "Close" button.
