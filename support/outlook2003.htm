Configuring Outlook 2003

The key configuration points are:

  • POP3 e-mail server type
  • Logon name your full e-mail address
  • POP3 server is "pop.readyhosting.com"
  • SMTP server is "smtp.readyhosting.com"
  • Outgoing server (SMTP) requires authentication

To create a new E-mail account select "E-mail Accounts..." from the "Tools" menu.

Select the "Add a new e-mail account" radio button and then select "Next >"

Select POP3 as the Server Type and then select "Next >"

Under the "User Information" section enter your name and your full e-mail address.

Under the "Server Information" section enter "pop.readyhosting.com" for your POP3 and "smtp.readyhosting.com" for your SMTP server.

Under the "Logon Information" section enter your complete e-mail address for the User Name and enter your Password.

Select "More Settings..."

Select the "Outgoing Server" tab.

Check the "My outgoing server (SMTP) requires authentication".

Make sure the "Use same settings as my incoming mail server" is selected.

Select "OK"

Select "Test Account Settings..."

Once it has completed, select "Close" on that dialog.

Select "Next >"
