Coding Guidelines
COM Guidelines
Files
Use standard file prefixes All header files (especially public files) should use standard prefixes.
Prefix Description
C Header files that declare the class (CLSID) GUIDs of COM objects that are to be used in the system.
I Header files that declare interfaces and the associated (IID) GUIDs.
U Header files that declare structures, enums, and constants.
